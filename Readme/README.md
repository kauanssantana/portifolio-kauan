## 📖 Sobre o Projeto

Este portfólio é minha principal vitrine digital. Ele foi projetado para oferecer uma experiência de usuário (UX) fluida e imersiva, apresentando quem eu sou, minhas competências técnicas e os projetos que desenvolvi. 

Diferente de portfólios convencionais, este projeto foi construído do zero com **Vanilla JavaScript**, focando em performance extrema, código limpo e micro-interações que elevam a percepção de qualidade do design, sem a dependência de bibliotecas ou frameworks pesados.

---

## ✨ Principais Funcionalidades (UX/UI)

* 🌓 **Dual Theme System:** Suporte nativo a temas Dark e Light, com persistência de preferência via `LocalStorage`.
* 🌎 **Internacionalização (i18n):** Suporte completo a dois idiomas (Português e Inglês) de forma dinâmica, sem recarregamento de página.
* ⌨️ **Efeito de Digitação:** Hero section com citação animada (Typing Effect) que se adapta ao idioma selecionado.
* 📸 **Interação 3D Tilt:** Foto de perfil com efeito de inclinação 3D baseada no movimento do cursor (Mouse Parallax).
* 🌊 **Reveal on Scroll (Intersection Observer):** Animações suaves de surgimento de conteúdo conforme o usuário navega verticalmente pela página.
* 🖱️ **Micro-interações de Hover:** Efeitos premium como "Sweep to Right" nos cards de competências.
* ✉️ **Formulário de Contato Real:** Integração direta para envio de mensagens para e-mail real utilizando FormSubmit / Netlify Forms.
* 📱 **Mobile First:** Layout 100% responsivo, utilizando Flexbox e CSS Grid para adaptação perfeita em qualquer tela.

---

## 💻 Tecnologias Utilizadas

A stack foi escolhida para demonstrar domínio absoluto das tecnologias base da web:

* **HTML5:** Estrutura semântica e acessível.
* **CSS3:** Variáveis nativas (`:root`), Flexbox, CSS Grid e animações complexas com `cubic-bezier`.
* **JavaScript (ES6+):** Manipulação avançada de DOM, Intersection Observer API, eventos de mouse e lógica de internacionalização.
* **Shields.io:** Geração de badges dinâmicos para a seção de habilidades.

---

## 📂 Estrutura de Seções

1. **Início (Hero):** Título de impacto, resumo profissional e citação animada.
2. **Sobre Mim:** Descrição detalhada da trajetória acadêmica (Senac) e missão profissional.
3. **Habilidades:** Grade interativa de competências e arsenal de ferramentas.
4. **Projetos:** Grid simétrico com cartões detalhados, tags de tecnologia e links de acesso/repositório.
5. **Contato:** Seção de networking com informações de localização, redes sociais e formulário funcional.
